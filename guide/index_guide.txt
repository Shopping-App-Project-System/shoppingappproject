==============================
index.html 主頁完整指南
==============================

【一、頁面用途】

此頁為「商店首頁」

負責：
1. 顯示品牌（store_name）
2. 顯示主視覺（hero）
3. 顯示商品列表（products）


----------------------------------------
【二、後端必須提供資料】
----------------------------------------

render_template(
    "index.html",
    store_name=...,
    hero=...,
    products=...,
    auth=...,
    footer_text=...
)

----------------------------------------
【三、hero（主視覺）】
----------------------------------------

hero = {
    "title": "夏季優惠",
    "subtitle": "全館8折"
}

👉 若沒給 → 使用預設文字


----------------------------------------
【四、products（商品資料）】
----------------------------------------

products = [
    {
        "name": "iPhone",
        "price": 30000,
        "description": "Apple 手機",
        "image_url": "/static/img/iphone.jpg",
        "cart_url": "/cart/add/1"
    }
]

----------------------------------------
【五、商品欄位說明】
----------------------------------------

name：名稱  
price：價格  
description：描述  
image_url：圖片  
cart_url：加入購物車 API  

----------------------------------------
【六、畫面邏輯】
----------------------------------------

✔ 有商品 → 顯示商品卡  
✔ 沒商品 → 顯示空狀態  

----------------------------------------
【七、圖片處理】
----------------------------------------

image_url 若為 None：

→ 使用 /static/default.png


----------------------------------------
【八、登入狀態（auth）】
----------------------------------------

auth = {
    "logged_in": True
}

👉 控制顯示：
登入 / 登出


----------------------------------------
【九、空狀態設計】
----------------------------------------

不是空白  
不是模板  

👉 是設計過的提示畫面


----------------------------------------
【十、設計原則】
----------------------------------------

✔ 簡單  
✔ 乾淨  
✔ 後端控制  
✔ 無模板痕跡  


----------------------------------------
【結論】
----------------------------------------

首頁的本質：

👉 展示商品 + 引導購買

==============================

==============================
member.html 完整教學指南
==============================

【一、頁面用途】

會員中心負責：

1. 顯示使用者資料
2. 顯示收件資訊
3. 顯示訂單紀錄


----------------------------------------
【二、後端提供資料】
----------------------------------------

render_template(
    "member.html",
    user=...,
    address=...,
    orders=...
)


----------------------------------------
【三、user（會員資料）】
----------------------------------------

user = {
    "name": "王小明",
    "account": "ming123",
    "email": "ming@test.com",
    "phone": "0912xxx",
    "level": "一般會員"
}


----------------------------------------
【四、address（收件）】
----------------------------------------

address = {
    "name": "王小明",
    "phone": "0912xxx",
    "full": "台北市..."
}


----------------------------------------
【五、orders（訂單）】
----------------------------------------

orders = [
    {
        "id": "ORD001",
        "total": 2000,
        "status": "done"
    }
]


----------------------------------------
【六、狀態對應】
----------------------------------------

done → 已完成  
pending → 處理中  
cancel → 已取消  


----------------------------------------
【七、畫面邏輯】
----------------------------------------

✔ 有資料 → 顯示  
✔ 沒資料 → 顯示空狀態  

----------------------------------------
【八、設計原則】
----------------------------------------

✔ 不寫死資料  
✔ 後端控制  
✔ 不露模板  
✔ 空資料也正常  

----------------------------------------
【結論】
----------------------------------------

會員中心是：

👉 使用者專屬資料頁
